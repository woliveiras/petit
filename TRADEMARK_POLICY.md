The source code in this repository is distributed under AGPL-3.0.
The Petit name, logo, and visual identity are trademark assets owned by the maintainer and are not automatically licensed with the code.

## Allowed Use

- Nominative reference in articles, reviews, and technical comparisons.
- Educational, research, and study use.
- Technical forks for local/private development that are not redistributed as official branded builds.

## Prohibited Use

- Publishing any redistributed app (commercial or noncommercial) using the Petit name (or a confusingly similar name) without explicit authorization.
- Publishing any redistributed app (commercial or noncommercial) using the official logo, icons, or brand identity without explicit authorization.
- Suggesting official affiliation, endorsement, partnership, or certification without permission.
- Repackaging this app for sale using original branding.

## Rules for Forks

Public forks must:

1. Use a distinct, original name.
2. Replace logos, icons, and brand text.
3. Change `applicationId`/package and publishing metadata.
4. Clearly disclose that the fork is unofficial.

Trademark use requires explicit authorization from the maintainer.
